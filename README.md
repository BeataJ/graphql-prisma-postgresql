### Steps to run

1. cd prisma
2. Start Prisma server: docker-compose up -d
3. Deploy Prisma: prisma deploy
4. stop Prisma server: docker-compose down

### environments

1. Deploy Prisma: prisma deploy -e ../config/dev.env(or prod.env)
